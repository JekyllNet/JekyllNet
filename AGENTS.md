# AGENTS

This file defines mandatory repository-level constraints for all AI agents operating in this repository.

## Mandatory Rules

- Use C# and .NET 10 only when building tools or automation for this repository.
- Do not use PowerShell (`ps` or `pwsh`) for tool implementation.
- Do not use any other scripting/runtime approach (for example Python, Node.js, Bash) for tool implementation.

## Exception Process

- If a task cannot be completed under these constraints, document the reason in a PR and obtain maintainer approval before proceeding.
