今日宣布，JekyllNet `v0.2.0` 正式发布。本次版本重点并非只在“新增功能点”，而在于把“可用能力”变成“可被快速掌握、可被稳定复用”的完整工具体系。

## 官方入口

- [JekyllNet 仓库](https://github.com/JekyllNet/JekyllNet)
- [JekyllNet 文档网站](https://jekyllnet.help)
- [JekyllNet GitHub Pages 入口](https://jekyllnet.github.io/JekyllNet/)

## v0.2.0 核心亮点

- 新增完整使用说明：从安装、构建、预览、翻译、发布到排错，一文直达
- 文档站分类梳理升级：按目标、按主题、按角色皆可定位
- 新闻与博客联动增强：版本动态可直接跳转到深度使用文档
- 中英文内容同步补齐：降低团队协作与对外传播成本

## 本次发布适合谁

- 想把 Jekyll 风格站点迁入 .NET 工作流的团队
- 想要 GitHub Pages 风格体验，但希望拥有更可控 CLI 与 CI 的项目
- 需要多语文档与 AI 翻译增量维护能力的内容团队
